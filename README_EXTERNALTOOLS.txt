
The symlink externaltools links to ../externaltools/externaltools

Checkout externaltools like this (cd to the parent directory of higgstautau
first so that externaltools is alongside higgspy):

svn co svn+ssh://${USER}@svn.cern.ch/reps/atlasphys/Physics/Higgs/HSG4/software/common/externaltools/trunk externaltool

See the README in externaltools for further instructions on how to checkout
and build all of the tools.
